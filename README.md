## Структура проекта

├── client/ # Vite + Vue 3 фронтенд
└── server/

## Установка

1. `git clone https://github.com/ваш-логин/vue-node-fullstack.git`
2. `cd vue-node-fullstack/client && npm install`
3. `cd ../server && npm install`

## Запуск

### Development

# В одном терминале

cd server && npm run dev

# В другом терминале

cd client && npm run dev
